> 计算机专业英语词汇学习记录，包含生词、短语及技术文档常用表达。

## 项目简介

本仓库用于记录计算机专业英语学习过程中遇到的不熟悉词汇、常用短语和技术文档中的典型表达。帮助提升阅读英文技术文档、论文和开源项目文档的能力。适合计算机专业学生和开发者系统学习专业英语。

## 学习内容概述

### 词汇分类

- **编程语言相关**：语法关键字、数据类型、控制结构等
- **数据结构与算法**：算法名称、复杂度术语、数据结构名词
- **操作系统**：进程、内存、文件系统等相关术语
- **网络技术**：协议、架构、安全相关词汇
- **数据库**：SQL、NoSQL、数据模型相关术语
- **软件开发**：设计模式、架构模式、开发方法论
- **前端技术**：HTML/CSS/JavaScript 相关术语
- **人工智能/机器学习**：AI/ML 领域专业词汇
- **云计算/DevOps**：容器、编排、CI/CD 相关术语

### 短语与表达

- 技术文档常用句式
- 代码注释规范表达
- Git 提交信息常用短语
- 技术博客写作表达

### 技术文档常用短语

#### 描述功能

- `is used to` - 用于...
- `allows you to` - 允许你...
- `enables...to` - 使...能够...
- `provides a way to` - 提供一种...的方法
- `is designed for` - 专为...设计

#### 说明限制

- `is not supported` - 不受支持
- `is subject to` - 受...限制
- `may result in` - 可能导致
- `is limited to` - 仅限于
- `requires...to` - 需要...才能

#### 给出建议

- `it is recommended to` - 建议...
- `you should consider` - 你应该考虑...
- `as a best practice` - 作为最佳实践
- `for better performance` - 为了更好性能
- `to avoid...` - 为了避免...

## 学习方法建议

1. **阅读官方文档**：优先阅读英文原版文档
2. **查看源码注释**：学习开源项目的英文注释
3. **写英文注释**：在代码中使用英文写注释
4. **记录生词**：遇到不熟悉的词汇立即记录
5. **定期复习**：使用间隔重复法巩固记忆

## 贡献指南

欢迎提交 Issue 和 Pull Request 来完善本仓库！

### 贡献方式

1. **补充词汇**：添加新的专业词汇
2. **修正错误**：指出文档中的错误
3. **分类整理**：帮助整理词汇分类
4. **添加例句**：为词汇提供实用例句

### 词汇格式

```markdown
### 单词

**音标**: /pronunciation/

**词性**: n./v./adj./adv.

**中文释义**: 中文解释

**例句**: 
- English example sentence.
- 中文翻译

**相关词汇**:
- synonym (同义词)
- antonym (反义词)

**常见搭配**:
- common phrase 1
- common phrase 2
```
